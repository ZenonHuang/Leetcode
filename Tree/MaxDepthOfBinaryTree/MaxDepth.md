# 求二叉树最大深度

给定一个二叉树，找出其最大深度。

二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。

说明: 叶子节点是指没有子节点的节点。

示例：
给定二叉树 [3,9,20,null,null,15,7]，

        3
       / \
      9  20
        /  \
       15   7

返回它的最大深度 3 。

### 思路

主要思路就是 **深度优先搜索**，DFS（Depth-First-Search ）.

要利用好**递归**：

终止条件是当前节点为空。

返回值：节点为空时说明高度为 0，所以返回 0；节点不为空时，则分别求比较左右子树的最大深度，同时加1表示当前节点的高度，返回该数值

时间复杂度：O(n)
